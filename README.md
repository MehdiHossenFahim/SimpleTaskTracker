# Simple Task Tracker 

A beginner-friendly Python console application that allows users to manage their daily tasks efficiently.  
This project demonstrates the use of **classes, file handling (JSON), and a menu-driven interface**.

---

##  Features

- Add new tasks with title and description
- View all saved tasks
- Delete tasks by selecting task number
- Automatic data saving using JSON file
- Persistent storage (tasks remain saved after closing the program)

---

##  Concepts Used

- Object-Oriented Programming (OOP)
- File Handling
- JSON Data Storage
- Exception Handling
- Menu-driven program design
